Question 1. Working with data.frame

Load the following data.frame, tvshows.

Show the code 
library(tidyverse)
tvshows <- read_csv(
        'https://bcdanl.github.io/data/tvshows.csv')

Q1a.

Provide the R code to return the shows whose Duration values are 30.

Show the code 
q1a <- tvshows[tvshows$Duration == 30, ]
Show the code 
q1a_indices <- which(tvshows$Duration == 30)
q1a <- tvshows[ q1a_indices, ]

The number of shows whose Duration is 30 is 13.


Q1b.

Provide the R code to return the shows whose GRP values are greater than the mean value of GRP in all the shows in the tvshows data.frame.

Hint: We can calculate the mean value of numeric vector x using the mean(x) function.

Show the code 
q1b <- tvshows[tvshows$GRP > mean(tvshows$GRP), ]

The mean value of GRP is 823.905.


Q1c.

Provide the R code to return the data.frame with only three variables—Show, PE, and GRP—for which PE values are greater than the mean value of PE.

Show the code 
q1c <- tvshows[tvshows$PE > mean(tvshows$PE), ]
q1c <- q1c[ , c("Show", "PE", "GRP")]
